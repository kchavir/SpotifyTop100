<h1>Top 100 Spotify Playlist</h1>



<h2>Description</h2>
This program creates a Spotify playlist, that shows top 100 popular songs on a selected date. <br />



<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>BeautifulSoup</b>
- <b>Musical Time Machine</b>
- <b>spotipy</b>
